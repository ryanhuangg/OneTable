OneTable
This library functions as a helper for users to build dynamic tables in websites. The library can serve many purposes such as taking notes, keeping track and managing data, or just simply putting ideas down. The library contains features such as adding or remove columns/rows, editing data, changing the look of the table, sorting the table, and summing or averaging data. The library also includes a import/export feature that allows users to easily save their ideas or to visualize existing data.
